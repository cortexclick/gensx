# Contributing

## ⚙️ Developing the library

### 📦 Building

```bash
pnpm build
```

### 🧪 Testing

If you want to run the tests of the project, you can execute the following command:

```bash
pnpm test
```

### 💅 Linting

To run the linter you can execute:

```bash
pnpm lint
```

And for trying to fix lint issues automatically, you can run:

```bash
pnpm lint:fix
```
